安装brew和node

  • 1.安装brew：官方指导地址 https://brew.sh/index_zh-cn

  • 2.检查brew是否已经安装：brew -v

  • 3.安装node：brew install node (注意版本，碰到新版本页面生成空白的问题)
  • 4.确认node版本： npm -v
  • 5.使用npm：官网地址 https://www.npmjs.com/
